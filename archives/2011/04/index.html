首先看看如下一个声明：

int ( ( fun )( int ) )[10];

get vtable test

好久以前看过这种东西,时间长了就忘了.复习下
class A{
public:
A():x(0){}
virtual void func1(){cout<<"vtable1";}
virtual void func2(){cout<<"vtable2";}
int x;
};
main中
A a;
int vtable = (int )&a;
void (p)(void);
p = ( void()() ) ((vtable)+0) ; // or this way : p = ( void()() ) ((vtable)[0]) ;注意[]和的优先级!!!
p();
p = ( void()() ) ((vtable)+1) ;
p();
最后vtable后面跟着数据,(这里x是private的也能获得) 强大啊-.-
cout<<( (int)(&a) +1 );
联想:
int i=0;
for(;;++i)
{
p = ( void()() ) ((vtable)+i) ;
}
想看别人的类设计了几个virtual function,这么测试看看崩在i是多少的时候,就知道了吧
.....真是2b想法....要是返回值和参数表不是void呢....
